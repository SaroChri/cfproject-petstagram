# cfproject-petstagram
<b>A project for pet adoptions</b>

<ul>Technologies used:
<li>Spring Boot</li>
<li>Java 11</li>
<li>MySQL</li>
<li>Thymeleaf</li>
<li>CSS</li>
</ul>

<p>In this project a user can create an account and upload a pet for adoption or pick one available to adopt</p>
